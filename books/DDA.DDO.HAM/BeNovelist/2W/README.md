      ... 작가에게 무엇보다 중요한 책무는 조금이라도 질 좋은 작품을 지속적으로 독자에게 제공하는 것**입니다. 나는 일단 현역 작가고, 말을 바꾸자면 아직 발전 도상에 있는 작가입니다**.
      문학이라는 이른바, 전쟁터의 최전선에서 맨몸으로 혈전을 펼쳐나가는 상황**입니다. **거기서 살아남고 또한 앞으로 나아가는 것, 그것이 내게 주어진 과제(task)**입니다.
    - **Inspired**
